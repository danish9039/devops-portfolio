Inside My CNCF LFX Mentorship with Jaeger

During the CNCF LFX mentorship, I worked on the Jaeger next-generation demo project. The focus was to build a practical reference architecture that combines observability concepts with production-ready deployment practices.

What I worked on

  • Designed architecture for an Oracle Kubernetes Engine setup integrating OpenTelemetry demo services, Jaeger collector, and OpenSearch.
  • Set up Kubernetes infrastructure and deployment patterns with Helm and Kustomize.
  • Integrated OpenTelemetry Collector for OTLP data routing and trace pipeline visibility.
  • Built CI/CD automation for environment deployment and state consistency.

Open-source contribution impact

I contributed auth transport improvements and token loader support to strengthen secure OpenTelemetry integration patterns. This work reached production environments used by over 5000 organizations.

Key learning

Mentorship accelerated my thinking around reliability by forcing architecture decisions to balance developer experience, operability, and long-term maintainability.
